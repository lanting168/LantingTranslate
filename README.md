# 🌐 多语言翻译工具

一款轻量高效的翻译软件，支持 **100+ 种语言互译**，可直接翻译 **PDF、Word、Excel、PPT 等文档**，并 **完整保留原始格式**。支持 **离线翻译** 与 **在线增强模式**，兼顾隐私、速度与准确性！

## ✨ 核心功能

- 🌍 **100+ 语言互译**：覆盖主流及小语种，满足全球化需求  
- 📄 **文档翻译**：支持 `.pdf`、`.docx`、`.xlsx`、`.pptx` 等格式  
- 🎨 **格式保留**：翻译后文档排版、样式、图表位置与原文一致  
- 📴 **离线翻译**：内置轻量语言模型，无网络也能快速翻译  
- ☁️ **本地模式**：完全离线运行，保障数据隐私，适合敏感内容
- 🖥️ **系统支持**：Windows
